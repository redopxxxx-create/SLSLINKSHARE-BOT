## ✨ ғᴇᴀᴛᴜʀᴇs
- **Unlimited Channels:** Jitne marzi channels add aur manage karein.
- **Fast Generation:** Normal aur Join Request links turant banayein.
- **Auto-Revoke:** Copyright se bachne ke liye links 5 min mein expire ho jate hain.
- **Auto-Approve:** Join requests ko automatic approve karne ka system.
- **Force Sub:** Users ko link tabhi milega jab wo aapka channel join karenge.
- **Hinglish UI:** Desi touch ke saath user-friendly interface.
- **External Links:** `/genlink` se kisi bhi bahar ke link ko encode karein.
- **Admin System:** Dynamic admin add/remove feature.
